# The Ayu Touch – Static Website

## 🛠 How to Host on GitHub Pages

1. Create a GitHub account and new public repo: `the-ayu-touch`
2. Upload all these files (do not change folder names)
3. Go to: Settings > Pages
4. Source = `main` branch and root (`/`)
5. Click Save

✅ Your website will be live at:
https://yourusername.github.io/the-ayu-touch
